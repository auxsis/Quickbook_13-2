Website Pathology Lab Management System

This module provides facility for customers to book a labtest along with the date and center directly from website. After the successful completion of test, the test report will be send to patient via email or the patient can also download the report from his account.

Basic Features

  • Provides facility to book a test directly from website.
  • Customers can see all the test of pathology and read its details.
  • Customers can add the test to cart to book it.
  • Admin can select Categories in labtest and test packages for easeness of customers.
  • Customers can select labtest and test packages according to Categories defined.
  • Admin can publish/unpublish both labtest and test packages to display test in website.
  • Facility of Email notification on confirmation of Test Request.
  • Facility of Email notification on completion of Test Report.
  • Also, test report can be send to patient via email.
  • Customer can view all the booked Test Request in their Account and download the report from their only.
